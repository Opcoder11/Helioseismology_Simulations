# Helioseismology Simulations

This repository contains Python scripts, Jupyter notebooks, and figures related to simulations of solar oscillations.  
The work demonstrates computational methods in helioseismology, including data visualization and analysis of oscillation modes.

## Repository Structure
- `scripts/` → Python scripts for simulations
- `notebooks/` → Jupyter notebooks with detailed analysis
- `figures/` → Plots and visualizations generated from simulations
- `fits/` → FITS files used in solar physics data analysis

## Requirements
- Python 3.10+
- NumPy
- Matplotlib
- SciPy
- Astropy (for FITS files)

Install dependencies with:
```bash
pip install numpy matplotlib scipy astropy
```
Run the main simulation script:
```
python scripts/visualize_fit.py
```
Or explore the Jupyter notebooks in notebooks/

Figures of Oscillation modes are available in figures/
